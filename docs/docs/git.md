# O que é Git
É um software que nos ajuda a fazer o versionamento dos projetos.

# Como instalar
Entre no site https://git-scm.com/install/ e faça o download e depois a instalção.

# Como utilizar o Git
Depois de instalaar o Git, os comandos git init para iniciar um projeto.

# Área de trabalho, área de preparação e repositórios no Git
No Git trabalhamos com etapa de projeto.
A etapa inicial é o que alteramos no nosso computador sem controle de versão que é a área de trabalho.
