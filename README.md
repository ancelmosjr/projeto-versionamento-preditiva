# Curso de Versionamento de Projetos - Peditiva

## Introdução
Este projeto serve como ilustração do curso da Preditiva de como trabalhar com versionamento de projetos com Git e GitHub.

## Objetivos

* Aprender usar VS Code
* Aprender sobre Poetry
* Aprender usar Git
* Aprender usar GitHub
* Aprender sobre documentação

## O curso

Durante o curso de versionamento de prjetos da Preditiva temos contato com diversas ferramentas

### VS Code
IDE utilizada para desenvolvimento dos projetos.

### Poetry
Pacote do Python utilizado para gestão dos projetos.

### Git
Ferramenta para controle de versões.

### GitHub
Repositório online para controle das versões dos projetos.
